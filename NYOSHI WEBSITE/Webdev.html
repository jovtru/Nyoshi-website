Advanced Safe Home Systems using Face-Recognition with Unique Passcode Systems

Abstract

In this paper, we present a model for an advanced security measure for home security systems using face-recognition and a unique passcode for each user. Both face recognition and unique passcode must be verified for the user to gain access to the building. For the face-recognition system, we use the HAAR cascade object detection algorithm with HAAR classifiers. For unique passcode, we use Google API for voice input submission and verification.

Keywords: Face recognition system, HAAR cascade object detection, HAAR classifier, Google, passcode

Introduction

The introduction section sets the context for the research, describing the importance of security in homes and the advantages of combining face and voice recognition technologies.

With increasing crimes, people seek robust security solutions for their homes. Facial recognition and voice recognition are two critical technologies for enhancing home security. Facial recognition maps and stores a person’s facial features as a faceprint, while voice recognition interprets spoken commands or dictation. Combining these technologies makes unauthorized access difficult. This system automates monitoring entry and exit times, reducing the homeowner's need to manually manage access.

Related Work

Numerous studies have focused on security systems, but few combine face and voice recognition effectively. Traditional methods like fingerprint and retina scanning are expensive and challenging to maintain. Our model offers high security with low cost and maintenance, utilizing a combination of facial and voice recognition.

Proposed Model

Our model integrates face-printing and voice-printing. When a guest arrives, the system captures their face sample and compares it with the database. If a match is found, the guest must provide a unique passcode. For technical issues or the owner’s absence, a one-time passcode can be used, which is sent via email.

Process:

  • Face recognition using a webcam and HAAR cascade algorithm.
  • Voice recognition via Google API.
  • Access granted upon matching both face and voice.

Pre-Processing

We use the ‘SpeechRecognition’ package and Google API for voice recognition. For face recognition, we use a web camera and HAAR cascade for object detection. The HAAR classifier processes frames from the camera.

Experiments

Dataset: Images and voice samples from 15 people were used. The system was tested under daylight and minimal ambient noise conditions.

Evaluation Metrics:

  • False Acceptance Rate (FAR)
  • False Rejection Rate (FRR)

Results: The model achieved 75-78% accuracy, with some failures due to lighting and noise.

Conclusion and Future Work

Conclusion: The proposed model enhances home security by combining face and voice recognition. The system reduces the homeowner’s manual efforts and provides high accuracy in access control.

Future Work: Future enhancements include integrating social media images for face recognition and improving system robustness against various environmental conditions.
